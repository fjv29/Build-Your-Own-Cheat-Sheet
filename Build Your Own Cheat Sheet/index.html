Colors In CSS

CSS Color Representation Color Unit & Range Example
Named Colors
  • Uses letters to spell names of colors that can be idenified by different browsers
  • Lime
  • Blue
  • Red
Hexadecimal/ Hex Colors
  • Starts with # and uses 6 or 3 digits
  • Uses 16 values represented from 0-9 then A-F
  • Lime: #00FF00 OR #0F0
  • Blue: #0000FF OR #00F
  • Red: #FF0000 OR #F00
RGB
(red, blue green)
  • Uses 3 values one for each red, blue, and green
  • Range for values is 0-255
  • rgb(0, 255, 0)[lime]
  • rgb(0, 0, 255)[blue]
  • rgb(255, 0, 0)[red]
HSL
(Hue, Saturation, Lightness)
  • Hue is measured in degrees from 0 to 360
  • Saturation is a percentage value ranging from 0% (gray) to 100% (fully saturated)
  • Lightness is also a percentage value, ranging from 0% (black) to 100% (white).
  • Lime: HSL(120, 100%, 50%)
  • Blue: HSL(240, 100%, 50%)
  • Red: HSL(0, 100%, 50%)
